[Mozilla 探索修改 Firefox UI](https://www.solidot.org/story?sid=66545)
------

<div>
Mozilla <a href="https://www.ghacks.net/2021/01/02/mozilla-is-working-on-a-firefox-design-refresh/">正在调查</a>修改 Firefox 的 UI，此举再次令很多 Firefox 用户<a href="https://news.slashdot.org/story/21/01/03/0516247/mozilla-is-working-on-a-firefox-design-refresh">感到担忧</a>。该项目被称为 Proton，Mozilla 已经在 Bugzilla 网站创建了名为“ <a href="https://bugzilla.mozilla.org/show_bug.cgi?id=proton" target="_blank">Proton - Firefox visual refresh</a>”的 meta bug 报告。Mozilla 考虑“现代化”多个 UI 元素，其中包括地址栏和任务栏、主菜单、信息栏、门钩、上下文菜单和弹出框。Mozilla 计划在今年年中 Firefox 89 发布时公布部分新设计。Firefox 89 的发布日期是 2021 年 5 月 18 日。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
