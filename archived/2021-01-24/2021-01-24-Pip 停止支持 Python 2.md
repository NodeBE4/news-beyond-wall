[Pip 停止支持 Python 2](https://www.solidot.org/story?sid=66746)
------

<div>
即将释出的 Python 包管理器 Pip <a href="https://pip.pypa.io/en/stable/news/#id1">即将释出 pip 21.0</a>，新版的一个重要变化是停止支持 Python 2，下载加入 --ignore-requires-python 支持，改进新解析器的错误信息显示，等等。Python 2 已经在 2020 年结束支持，开发者在 2020 年 4 月<a href="https://www.solidot.org/story?sid=64175">发布</a>了 Python 2.7 分支的最后一个版本也是一个非常特殊的版本：Python 2.7.18（近似欧拉常数）。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
