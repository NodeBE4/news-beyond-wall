<p>自鸿海集团创办人郭台铭上月底宣布有意通过旗下永龄基金会购买500万剂德国BioNTech（简称BNT）冠病疫苗供台湾使用，时至今日已过去大半个月，但这笔交易能否达成仍是一个问号。</p><p>这个问号的背后不仅是郭台铭和厂商之间的沟通合作程序，更大的不确定性还来自民进党政府的种种考量。　</p><p><strong>郭董深夜发文引人遐想</strong></p><section id="imu"><div id="dfp-ad-imu1">        </div></section><p>郭台铭今天凌晨零点左右在个人面簿上发表了一份声明，表示目前向台湾卫福部申请疫苗专案输入一事，正按计划进行中。</p><p>他在声明中强调，“所有的媒体报道，只要是未经我本人或永龄基金会证实，全都是与事实不合，不管是媒体臆测，或是未经我方证实之消息来源，其目的或许是在阻挠取得疫苗一案的进行……央求社会大众不要听信这些未经证实的消息。”</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/shenyeshenming_0.jpg?iTqbqNO7UyRedew6H07VocBBYhBKG5nA&itok=bwWYUUvK" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>郭台铭凌晨发声明，对陈时中和卫福部表示感谢。（面簿）</figcaption></figure>                  <p>在声明的最后，他感谢卫福部及其同仁在端午节假期间，牺牲假日与他及团队一起开会讨论相关细节，并特别感谢了卫福部长、台湾中央流行疫情指挥中心指挥官陈时中在星期日深夜还亲自致电，承诺将全力协助此次疫苗专案输入。</p><p>郭台铭这则声明发出不到半小时，就获得两万多名网民的点赞，有网民甚至在下面留言说，下次要选他做总统。</p><div id="innity-in-post"></div><div id="dfp-ad-midarticlespecial">        </div><p>但事情还没完，声明发出不久，永龄基金会执行长刘宥彤在凌晨1时28分留言回应：“就像是看见隧道那一端的光，但行进间忽隐忽现，要如何面对呢？”</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/liushenyehuifu1_1.jpg?jrXFYC1IfTz82.BrTw_ZCs8L0vgz3nES&itok=JpT6ryzT" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>永龄基金会执行长刘宥彤在郭台铭发出声明后很快留言回应。（面簿截取）</figcaption></figure>                  <p>而郭台铭竟在六分钟后迅速回复她说：“是不是出口，走下去才知道。”</p><p>两人深夜一问一答不禁让人浮想联翩，所谓的“光”和“忽隐忽现”到底在暗示什么？而郭董这个若有似无的回复又是否暗藏玄机呢？</p><p><strong>台湾资深媒体人解读声明：就是提醒陈时中　</strong></p><p>资深媒体人黄𬀩瀚在自己的一档节目中指出，郭董的这则声明其实是想提醒陈时中：“不要忘记您的承诺”，并划出了声明全文的重点：“感谢陈部长在周日深夜还亲自致电与我，承诺将全力协助此次疫苗专案输入”。</p><p>黄𬀩瀚说，这是郭台铭掀了一张底牌，让外界知道，原来陈时中星期日深夜曾有电话承诺全力帮忙。</p><p>除了提醒陈时中不要忘了之前的支持承诺，黄𬀩瀚分析认为，声明的另一层用意，是当郭台铭“公开感谢”陈时中后，陈时中或许就不好再对BNT疫苗进口一事横加阻扰或刁难。</p><p>换句话说，这或在暗指民进党很可能曾“表面支持”但“私下阻挡”，郭台铭此时作出表态，是想用“捧”的方式换取民进党的“不阻挠”，最终能使BNT疫苗进口计划顺利实现。</p><p>黄𬀩瀚还发现，郭台铭对刘宥彤最开始的回复是：“是不是出口，要走下去才知道”，而最后决定去掉这个“要”字，前后用了三分钟，显示他内心的纠结，也被黄𬀩瀚解读为郭台铭是希望传达“更加正向”的信息。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/guotaiming_jiujie__0.jpg?E2ad9y9uwP1kS17.4UGFHTLr8Di8ANJf&itok=q6Iw8P29" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>郭台铭发出回复三分钟后又删掉了其中一个字（视频截取） </figcaption></figure>                  <p><strong>民进党的如意算盘</strong></p><p>对于郭台铭的声明，黄𬀩瀚质疑，既然陈时中星期日才表示了支持，为何周二（15日）就有媒体独家爆料德国BNT总厂不喜欢跟郭台铭谈合作，并认为他应该退出这次疫苗采购这样的新闻？</p><p>黄𬀩瀚指的是《新新闻》周二标题耸动的独家新闻《“其实BNT不想跟郭台铭谈！”德国原厂高层大爆国际疫苗谈判秘辛》。</p><p>该文指出“BNT不想谈”的两个原因：一是担心郭台铭太过政治性，觉得没有安全感；二是郭台铭会透过媒体、民意和投资人来施压，谈判风格过于强势。</p><p>言外之意是，BNT疫苗引进台湾迟迟不能成事，根源不只是卫福部要求“原厂授权书”，而是还有郭台铭个人因素。</p><p>黄𬀩瀚在头一天的节目中就质疑该文说，如果德国原厂真的不想跟郭台铭打交道，那找一个不被原厂信任的人去谈买疫苗干嘛？称“郭台铭现在就可以撤了”。</p><p>他还对文章其他多个论点表达了不解，分析这是有人不想让郭台铭把功劳整碗端走，所以才传出其实BNT总厂不喜欢郭台铭，不想跟郭台铭谈，降低郭台铭的贡献度，把郭台铭讲得可有可无。</p><p>这或许正是民进党打的如意算盘：往郭董身上泼点脏水，即如果疫苗最终来不了，则少一桩心事；但如果真的“不幸”成事了，民进党至少还能在引进疫苗的功劳中分得一大杯羹，捞取不少政治资本。</p><p><strong>郭台铭的疫苗还有戏吗？</strong></p><p>无论是“私下阻扰”还是“明面抢功”，民进党政府在这场政治博弈中可谓费尽心机。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/guotaimingfaxin_0.jpg?kKjPcdp3aET2t3sTj35lSm5gzotxCJir&itok=4IZhgr7U" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>郭台铭曾在2019年宣布参加2020年总统选举国民党内初选，但在9月17日最后登记独立参选的前一天深夜，宣布决定不参与2020年连署竞选总统。（法新社）</figcaption></figure>                  <p>黄𬀩瀚就指出，郭台铭曾经和韩国瑜参加过总统初选，所以不能排除他有可能是下一届的台湾总统选举参选人，如果让民众打到由他引进的疫苗，“那还得了？”“打到郭台铭这500万剂疫苗的人下次可能就‘自动内化’，票投郭台铭！”</p><p>黄𬀩瀚的分析听起来或许有点耸动，但以民进党政府一个月来屡被质疑的抗疫表现，其面临的严峻政治形势也越来越明显。</p><p>对于疫苗供应问题，台北市长柯文哲上月27日喊话蔡英文政府“不要意气之争、不要政治操作、不要为了面子”，直言“老百姓生命才是第一考量”。</p><p>就连被称为“英粉”的深绿人士、台湾中研院院士陈培哲都对民进党政府的疫苗操作感到失望，指政府并不信任科学，其防疫决策的根据是政治，“若早点尊重专家意见，很多生命都不必牺牲”。</p><p>许多“老民进党人”也对蔡英文政府所作所为看不下去。民进党前主席施明德受访时表示，因为政府的“不作为”导致愈来愈多人命的逝去，质问执政者能无愧乎？</p><p>他还在面簿上写道：“人命关天，忠言逆耳”，沉重呼吁当政的“新民进党人”稍稍谦虚倾听。</p><p>另一名“老民进党人”、曾担任文宣部主任的知名媒体人陈文茜，形容这是第一次感觉有一个比铁石还要硬心肠的政府，痛批政府鄙夷地看着恐惧的子民......每天只看自己的民调，在疫情蔓延下想的是下一届选举。</p><p>的确，民进党在疫苗问题上一直都卡在“政治因素”。上海复星跟德国BNT药厂签约，拿下了BNT疫苗在海峡两岸四地的大中华地区独家代理权，不忿“被代理”的民进党政府自然想绕开复星，于是直接去跟德国BNT原厂谈，但进展不顺，被陈时中形容为“不是因为合约内的问题，而是合约外的问题”。</p><p>而另一边厢，永龄基金会执行长刘宥彤5月27日就曾表示，中国大陆及国台办没有干涉郭台铭购买疫苗的计划，称上海复星、德国BNT的疫苗是完全一样的，双方与美国辉瑞之间是属于商业合作模式，早在美国辉瑞之前，上海复星就已经在这个合作案里面了，强调“跟政治没有太大的关联性”，并说“政治要为经济服务”。</p><p>彼时，还在等待阿斯利康和莫德纳2000多万支疫苗能如数到货的蔡英文则说，疫苗采购需由中央统筹。</p><p>但没多久，面对台湾舆论及民意反弹，蔡英文5月31日发表谈话，感激台湾民间的热心帮忙，并宣布建立政府与民间对话机制，但因手续繁复仍被外界认为政府并不乐见民间买成疫苗，打乱他们重仓押宝的台产疫苗计划。</p> <p>就在今天，蔡英文再次对“民间援助”释放信号，她特地提到郭台铭的深夜声明，感谢郭台铭的帮忙，同时也解释说：疫苗进口不是去卖场采购，过程复杂，还牵涉多边的国际谈判，也充满国际政治的角力，力图澄清“政府阻挡民间采购疫苗”的传闻。</p><p>暂时还不确定蔡英文的上述表态是否是对郭台铭进口疫苗放行的迹象，倘若果真如此，对台湾民众来说确属令人鼓舞的好消息。</p><p>不过，陈时中今天回应郭台铭声明时也表示，当前买疫苗非常困难，感谢民间团体还有郭台铭协力，公私协力一起努力，但不管如何，都非常困难，因国际疫苗现货不稳定，且大疫苗厂都跟国家往来。</p><p>这么一说，又似乎回到原点……</p><p>只是在大流行肆虐、人命关天的时候，仍在玩政治的人，恐怕得小心最终被政治反噬。</p>
