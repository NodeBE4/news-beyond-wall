[微软例行更新修复正被利用的 IE 漏洞](https://www.solidot.org/story?sid=65230)
------

<div>
微软本周释出了例行安全更新，修复了至少&nbsp; 120 个安全漏洞，有 17 个被归类为高危漏洞。其中 编号为 <a href="https://portal.msrc.microsoft.com/en-US/security-guidance/advisory/CVE-2020-1380">CVE-2020-1380</a> 的 IE 漏洞正被利用，用户使用 IE 浏览器访问恶意网站就可能被利用该漏洞执行任意代码入侵系统，并拥有当前用户的相同权限，如果是系统权限那么攻击者就能完全控制系统；另一个正被利用的漏洞 <a href="https://portal.msrc.microsoft.com/en-US/security-guidance/advisory/CVE-2020-1464">CVE-2020-1464</a> 允许攻击者绕过 Windows 的安全功能加载错误的签名文件；Windows Server 漏洞  <a href="https://portal.msrc.microsoft.com/en-US/security-guidance/advisory/CVE-2020-1472">CVE-2020-1472</a> 允许未经授权的攻击者获得域控制器的管理访问权限；Windows Print Spooler 漏洞 <a href="https://portal.msrc.microsoft.com/en-US/security-guidance/advisory/CVE-2020-1337">CVE-2020-1337</a> 允许攻击者或恶意程序进行提权。Adobe 本周也释出了<a href="https://helpx.adobe.com/security/products/acrobat/apsb20-48.html">安全更新</a>，修复了旗下产品的多个漏洞，其中包括 Acrobat 和 PDF Reader 的高危漏洞。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
