[《失忆症》系列作品公开源代码](https://www.solidot.org/story?sid=65631)
------

<div>
恐怖游戏《失忆症》系列的开发商 Frictional Games <a href="https://frictionalgames.com/2020-09-amnesia-is-now-open-source/?fbclid=IwAR0g0F4qr_znQbGeYkQs11MU1Vp2a9zDKwrbFeoTFfz0M-ZRD306GK4RaqI">宣布</a>公开《<a href="https://github.com/FrictionalGames/AmnesiaTheDarkDescent" target="_blank">失忆症：黑暗后裔</a>》和《<a href="https://github.com/FrictionalGames/AmnesiaAMachineForPigs" target="_blank">失忆症：猪猡的机器</a>》的完整源代码，代码托管在 GitHub 上，采用GPL v3 许可证。开发商表示开源并不意味着游戏免费，而是意味着可以在 GPL v3 下自由的使用源代码，free 是“free speech（言论自由）”的“free”而不是“free beer（免费啤酒）”的“free”。开发商的《失忆症》系列新作《<a href="https://store.steampowered.com/app/999220/Amnesia_Rebirth/" target="_blank">失忆症：重生</a>》将在一个月后发售，它的另一款知名作品是科幻恐怖游戏《SOMA》。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
