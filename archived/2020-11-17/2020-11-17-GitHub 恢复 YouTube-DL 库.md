[GitHub 恢复 YouTube-DL 库](https://www.solidot.org/story?sid=66113)
------

<div>
GitHub <a href="https://github.blog/2020-11-16-standing-up-for-developers-youtube-dl-is-back/">宣布恢复</a> YouTube-DL 库。上个月，GitHub 在收到美国唱片业协会（RIAA）的 DMCA 通知后移除了 YouTube-DL 库。RIAA 给出的侵权理由是 YouTube-DL 库包含了下载版权音乐的测试范例。此事引发了广泛关注，电子前哨基金会向 GitHub 递交了<a href="https://github.com/github/dmca/blob/master/2020/11/2020-11-16-RIAA-reversal-effletter.pdf">正式文件</a>，指出 YouTube-DL 的做法事实上并没有构成对 DMCA 法律的违反。YouTube-DL 项目的维护者也递交了补丁移除了相关的示例，因此 GitHub 决定恢复 YouTube-DL 库，它同时宣布将向开发者防御基金捐赠 100 万美元，帮助收到此类法律文件的开发者。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
