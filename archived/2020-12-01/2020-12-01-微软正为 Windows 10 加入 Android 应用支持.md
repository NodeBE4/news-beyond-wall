[微软正为 Windows 10 加入 Android 应用支持](https://www.solidot.org/story?sid=66245)
------

<div>
微软据<a href="https://www.windowscentral.com/windows-10-project-latte-android-apps">报道</a>正为 Windows 10 加入 Android 应用支持。该项目被称为 Project&nbsp;Latte，将允许应用开发商无需改变代码或只要略微改一下代码就能将 Android 应用带到 Windows 10 上。在 Windows 上运行的 Android 应用将打包成 <a href="https://docs.microsoft.com/en-us/windows/msix/overview"> MSIX 文件</a>，可以通过 Windows Store 发行。Project Latte 将基于现有的 Windows Subsystem for Linux (WSL) 子系统。WSL 是 Windows 10 平台上的虚拟化 Linux 内核，让使用 Windows 命令行运行 Linux 应用成为可能，而 Android 也是运行在 Linux 内核上的。为了支持 Android 应用，微软需要实现它的 Android Runtime (ART)版本。微软预计将在明年宣布 Project&nbsp;Latte，预计它将会随 2021 年秋季更新释出。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
