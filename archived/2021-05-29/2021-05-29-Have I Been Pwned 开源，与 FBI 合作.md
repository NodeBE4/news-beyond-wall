[Have I Been Pwned 开源，与 FBI 合作](https://www.solidot.org/story?sid=67900)
------

<div>
Have I Been Pwned 维护者 Troy Hunt <a href="https://www.troyhunt.com/pwned-passwords-open-source-in-the-dot-net-foundation-and-working-with-the-fbi/">宣布</a>网站代码在 .NET 基金会的帮助下开源，<a href="https://github.com/HaveIBeenPwned">源代码</a>发布在 GitHub 上，采用 BSD 3-Clause 许可证。 .NET 基金会不隶属于微软，而是一个独立的非盈利基金会。Troy Hunt 还同时宣布与美国执法机构 FBI 合作。FBI 对网络犯罪分子的调查毫无疑问会涉及到大量的密码，如何处理这些被犯罪分子窃取和利用的密码？FBI 接触了 Troy Hunt，决定向 Have I Been Pwned 提供密码的 SHA-1和 NTLM 哈希对，网络用户将可以通过检查网站了解自己的密码是否遭到泄露。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
