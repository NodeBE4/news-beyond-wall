<p>中国各省份近日将陆续公布高考成绩及分数线，大学也将陆续放榜并寄发录取通知书。今年各大学的录取通知书继续发挥创意，给新生带来惊喜。</p><p>综合《潇湘晨报》《新京报》等中国媒体报道，今年全新的清华大学立体录取通知书已制作完成，等待填写信息寄送。</p><p>从2018年起，清华大学的立体二校门录取通知书即红遍网络。随后，清华大学每年的录取通知书均延续了“立体二校门”设计。</p><section id="imu"><div id="dfp-ad-imu1">        </div></section>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/1c6a476916d84d95ab94b37b3f939137.jpeg?K7LbKvza_nSzCVN9CfXOQth3BX3Tr.VL&itok=6hfd9tZA" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>清华大学的立体二校门录取通知书意在祝福新生“破壁追光、圆梦清华”。（互联网）</figcaption></figure>                  <p>据悉，今年清华大学录取通知书的外壳材质和立体二校门都进行了全新升级，激光雕刻、手工拼插的新版通知书颜色更加亮丽持久，立体结构更加稳固，意在祝福新生“破壁追光、圆梦清华”</p><p>哈尔滨工业大学今年发布了“发光”录取通知书，采用镭射材质，在光线的照射下可以看到颜色的变化。通知书共有四个部分，除了通知的功能，还包含感谢信和感谢信的相框，让学生可以把这封感谢信，赠与自己的挚友或者是父母。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/641.jpg?vwYgLvM34vhjPNGlDNyLaYK4pF9CXm12&itok=k9YZ5uSn" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>哈工大的“发光”录取通知书在光线的照射下可以看到颜色的变化。（互联网）</figcaption></figure>                  <p>南开大学的录取通知书以校色青莲紫做底色，拉动内页就合拢日月两页。随通知书一并寄出的还有一个荷包，其中有两粒来自浙江嘉兴的莲花种子，激励学生在新生代乘风破浪、扬帆远航；此外还包含一枚“不负韶华”的书签，一封90岁工程院院士、南开大学毕业的第一名研究生李正名先生的信。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/250df1852f89455398f676c1916f1b79.jpg?kn1Ev6RBRt2OwHXpq2CE1CoLdbrZSEWC&itok=R_jTnrvt" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>南开大学随通知书一并寄出的还会有一个荷包，其中有两粒来自浙江嘉兴的莲花种子。（互联网）</figcaption></figure>                  <p>同济大学的录取通知书则附带一份“造船指南”，新生可折纸叠成同舟号、共济号和梦想号三艘小船，寓意新生“扬起梦想的风帆，同舟共济，乘风破浪。”</p><div id="innity-in-post"></div><div id="dfp-ad-midarticlespecial">        </div><p>通知书还附上有大学四年200周倒数周记的时光笔记，希望新生做好规划，不负韶华。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/download_0.jpg?OzZRCOyVVb7VJAQCsi0oKyTu1a_8XmHE&itok=fMdSajmT" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>同济大学的通知书附带一份“造船指南”，新生可折纸叠成三艘小船。（互联网）</figcaption></figure>                  <p>上海大学则为本科新生送上一颗“未来芯”，启动“未来芯”钥匙校徽背后的按钮，便可点亮“未来芯”，将其靠近手机即可读取微信小程序“上大锦SHU”（音同“上大锦书”），新生能够第一时间观看“红色传承”宣传片，同时获取个人专属电子录取通知书。</p>      <figure class="image-placeholder "><img alt class <figcaption>上海大学则为本科新生送上一颗“未来芯”。（互联网）</figcaption></figure>                  <p>天津大学今年的星空版通知书也引发关注，有一颗位于火星与木星之间的8917号小行星，被命名为“天津大学星”，今年通知书的开合处就有一颗可温感变色的“天津大学星”，色彩变化的刹那，寓意着学子与天大签订了一份“星河之约”。</p>
