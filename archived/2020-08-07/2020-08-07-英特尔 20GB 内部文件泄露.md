[英特尔 20GB 内部文件泄露](https://www.solidot.org/story?sid=65176)
------

<div>
英特尔<a href="https://www.zdnet.com/article/intel-investigating-breach-after-20gb-of-internal-documents-leak-online/">正在调查</a> 20GB 内部文件被<a href="https://mega.nz/folder/CV91XLBZ#CPSDW-8EWetV7hGhgGd8GQ">上传到 MEGA</a> （仅供参考）的安全事故。泄露的部分文件被标记为“机密级”或“限制级”。瑞士软件工程师 Till Kottmann <a href="https://twitter.com/deletescape/status/1291405688204402689" target="_blank">声称</a>这些文件是一位匿名黑客今年早些时候入侵英特尔获得到的。泄露的文件包括了 Git 代码库、开发工具等，部分密码加密的压缩文档使用了非常弱的密码，其中包括 Intel123 或 intel123。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
