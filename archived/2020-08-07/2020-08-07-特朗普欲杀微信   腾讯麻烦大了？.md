[特朗普欲杀微信   腾讯麻烦大了？](https://www.rfa.org/mandarin/yataibaodao/junshiwaijiao/rc-08072020125834.html)
------

<div>
<p>除了TikTok，美国总统特朗普周四晚间签署的行政命令，还包括将禁止美国企业与不论国籍的在美个人与微信及其母公司腾讯交易往来。虽然这项行政命令可能涉及的范围仍有待相关部门厘清，但消息一出，美国的游戏玩家担忧，微信全资持股的“英雄联盟”是不是不能玩了？更有分析人士指出，比起“字节跳动”可能被逼低价抛售TikTok，特朗普剑指微信对腾讯的杀伤力更大。究竟和腾讯有业务往来的美国企业还有哪些？</p><p>按住说话、松手随即就能把信息传给朋友，这个声音是不论身处何地的微信使用者再熟悉不过的了。不过，特朗普行政命令一出，在美国华人社区议论纷纷的就是：以后微信及微信相关的产品，例如微信支付，在美国还能不能使用？<br/> <br/> 细看统特朗普签署的行政命令，四十五天后、也就是从9月20日开始，在美国的实体，包括国际组织在内，以及在美国的任何个人，都禁止与微信（WeChat）及其母公司腾讯交易。</p><p>一位在美国读博士、不愿透露真实姓名的中国留学生告诉本台，他并不担心微信不能使用，也不同情一直以来做中共内容审查帮凶的微信，若微信未来真遭禁用，他和国内家人联络还有其它选择，例如小米的米聊平台。但是，如果美国对微信支付和支付宝也下重手，他认为，中国留学生在美国的生活会受到影响：<br/> <br/> “如果把微信支付或是支付宝也禁掉的话，这才会很不方面，因为在留学生圈子中，这很常用。”</p><p><img alt="rc0807a.jpg" class="image-inline" src="https://www.rfa.org/mandarin/yataibaodao/junshiwaijiao/rc-08072020125834.html/rc0807a.jpg" title="rc0807a.jpg"/></p><p><b>禁与微信交易 美国剑指中国支付系统？</b></p><p>以纽约为例，中国城内许多餐厅都提供微信支付或支付宝付款方式，对访美的中国游客与中国留学生来说，是非常方便的金融服务。然而，这样的移动支付服务，美国的金融业者等于完全被排除在外，而在中国的移动支付市场，美国业者却没能享有同等市场准入。<br/> <br/> 以美国的第三方支付Paypal为例，中国人民银行直到2018年才允许PayPal这类外国第三方支付企业在中国经营业务，但中国Paypal帐户只能在国内互通，不能转钱给国际版Paypal帐号。<br/> <br/> 美国禁止与微信及腾讯交易，涉及范围有多大？还有待四十五天后，以商务部为首的相关部门定夺。</p><p>对美国民众来说，电子游戏玩家则是关切还能不能连线玩游戏。<br/> <br/> 全球热门的线上游戏“英雄联盟”，他的开发商拳头公司（Riot Games）总部位于加州，但已由腾讯全资持股。美国的英雄联盟玩家以后购买宝物升级，算不算是与腾讯有交易？又是否在美国本次行政命令的限制中？<br/> <br/> 白宫一名不具名的资深官员告诉《洛杉矶时报》，这次行政命令不包涵腾讯持股的线上游戏类。</p><p> </p><ul><li><b><a 美国政府对微信与腾讯的不满，也在总统的行政命令中一目了然。<br/> <br/> 特朗普政府指出，微信据报会对中国共产党认为敏感的信息执行内容审查，甚至还成为中共假信息宣传行动的工具；微信撷取大量使用者资料，甚至取得美国公民个人资讯，还监控在美国的中国公民，美国必须像澳大利亚与印度一样，在限制或禁止使用微信上，采取更积极行动，以维护国家安全。<br/> <br/> 对于美国的指控，腾讯公司一名发言人在回复本台查询时仅表示，“腾讯正审视这次（美国总统的）行政命令，以全面了解相关资讯。”<br/> <br/> 这名发言人还另外附上了相关事实说明文件称，腾讯2006年在美国营运以来，致力于保护与腾讯合作的美国公司的知识产权与内容创意，腾讯在美国的合作伙伴多集中在影视娱乐产业，包括：华纳音乐、环球音乐、20世纪福斯、环球影业、HBO、美国职篮（NBA）、美国职棒大联盟（MLB）与美国职业橄榄球大联盟（NFL），都是腾讯的合作或战略合作伙伴。<br/> <br/> 文件内容还提到，腾讯在美国创造就业岗位至少两百五十个，在美国东、西两岸四个城市设有办公室。</p><p>对于白宫质疑微信内容审查的指控，腾讯则没有解释。</p><p><div class="image-inline captioned" style="width:680px;"><div style="width:680px;"><img alt="位于北京的微信母公司腾讯总部（美联社）" <br/></p><p>华盛顿智库美国企业研究所（AEI）学者史剑道透过电子邮件告诉本台，美国国务卿蓬佩奥（Mike Pompeo）周三就已经预告，美国认定微信是不可信赖的软件，除了Tik Tok，美国也扩大了自身行动的范围，如果美国不这么做，其它中国企业就可能继续在美国从事像字节跳动与腾讯的作为。<br/> <br/> 美国杀鸡儆猴，人在纽约、长期关注中国科技与经济问题的秦鹏则认为，现在处理微信的方式和当时美国在科技设备大厂先针对中兴，但主要目标其实是要咬着华为，有异曲同工的相似处。<br/> <br/> 他说：“我是觉得，如果打腾讯可能会打得更狠，实际上微信对世界的危害与威胁是很大的。”<br/> <br/> 秦鹏说，微信对在美国的使用者执行内容审查，甚至随便封停美国使用者的帐号，这都是明摆着的事。美国现在的态度就是，中国政府与企业说什么，都不相信了。<br/> <br/> 美国这次对微信与腾讯出手，一旦四十五天后禁止交易的命令严格落实，会如何影响美国企业与腾讯的合作及在中国的发展？截至发稿，NBA并没有回复本台查询。</p><p> </p><p>自由亚洲电台记者郑崇生华盛顿报道    责编：何平
