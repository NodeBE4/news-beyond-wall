[MIPS 投奔 RISC-V](https://www.solidot.org/story?sid=67144)
------

<div>
2018 年，Wave Computing 从 Imagination Technologies 收购 MIPS Technologies，<a href="http://hdardware.solidot.org/story?sid=58985">宣布</a>将在 2019 年第一季度发布 core R6 时开源。2019 年初它发起了 MIPS Open Initiative 倡议，但在 2019 年 11 月它终止了这一倡议。2020 年 4 月，Wave <a href="https://www.solidot.org/story?sid=64199" target="_blank">申请破产保护</a>。2021 年 3 月，Wave 更名为 MIPS，宣布将放弃开发 MIPS 架构，<a href="https://www.solidot.org/story?sid=64199">投奔</a> RISC-V 阵营。RISC-V 和 MIPS 都是基于精简指令集（RISC）原则，但今天的 MIPS 公司只是挂了一个名字，它表示会继续履行重组前签署的授权协议。MIPS 已经加入了 RISC-V 基金会，而 RISC-V 基金会的 CTO Mark Himelstein 曾是 MIPS Technologies 的前雇员。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
