[Rust 语言正吸引更多项目使用](https://www.solidot.org/story?sid=66436)
------

<div>
Rust 语言项目公布的<a href="https://blog.rust-lang.org/2020/12/16/rust-survey-2020.html">最新调查</a>显示，83% 的回应者表示<a href="https://www.zdnet.com/article/rust-programming-language-were-using-it-for-bigger-projects-say-developers/" target="_blank">正在使用</a> Rust 从事开发。Rust 是由 Mozilla 创造的语言，Mozilla 用它开发了实验性的渲染引擎 Servo，并逐渐将部分组件整合到 Firefox 中，但在前不久的裁员中 Mozilla 裁掉了许多 Rust 项目的核心开发者。Rust 项目宣布独立，而 Servo 项目则成为了 Linux 基金会<a href="https://www.solidot.org/story?sid=66121">托管项目</a>。微软和亚马逊 AWS 部门的工程师都在尝试使用 Rust 项目开发更安全的软件。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
