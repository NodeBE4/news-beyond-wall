[Perl  项目治理新规](https://www.solidot.org/story?sid=66216)
------

<div>
Perl 语言项目的新治理模式<a href="https://github.com/Perl/perl5/pull/18357/files">已接近成型</a>。 Python 项目的治理模式与 Python 项目采用的模式相似，将由两个实体参与治理：一个由几十人组成的核心团队，以及一个三人组成的指导委员会。核心团队制定规则，投票选出两个团队的成员，将实质性的决策权授予指导委员会。指导委员会有着广泛的权力，可以对语言、解释器和其它组件的开发做决定。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
