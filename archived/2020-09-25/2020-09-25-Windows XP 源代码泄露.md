[Windows XP 源代码泄露](https://www.solidot.org/story?sid=65646)
------

<div>
有近二十年历史 Windows XP 尚未完全退出历史舞台，统计显示仍然有 1% 的用户使用 Windows XP——即使用人数仍然有数百万之多。本周，4chan 论坛的用户<a href="https://www.gizmodo.com.au/2020/09/looks-like-the-windows-xp-source-code-just-leaked-on-4chan/">发布</a>了所谓的 <a href="https://anonfiles.com/Ffn8u2Yaoa/Microsoft_leaked_source_code_archive_2020-09-24_torrent">Windows XP SP1源代码</a>（种子，未验证）。独立研究人员表示它是可信的。源代码的公开意味着更容易找到漏洞，而这对早已结束支持的操作系统而言意味着将会面临更高的安全风险。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
