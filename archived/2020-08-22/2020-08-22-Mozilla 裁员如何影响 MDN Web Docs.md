[Mozilla 裁员如何影响 MDN Web Docs](https://www.solidot.org/story?sid=65322)
------

<div>
Mozilla 最近宣布裁员大约 250 人，很多重要项目受到影响，其中之一是 MDN Web Docs 服务。官方博客<a href="https://hacks.mozilla.org/2020/08/an-update-on-mdn-web-docs/" target="_blank">详细解释了</a>这次裁员或重组对 MDN 的影响：暂停支持  DevRel、<a href="https://hacks.mozilla.org/">Hacks blog</a> 和 <a href="https://wiki.mozilla.org/TechSpeakers" target="_blank">Tech Speakers</a>，因为裁员部分项目缩小规模，包括 Mozilla 开发者程序、开发者活动和 MDN 技术写作。Mozilla 的  Contributor Experience 总监 Rina Jensen 强调，MDN 并不会消失，他们正在制定计划保持内容更新，继续改进平台，包括基于 GitHub 的递交系统。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
