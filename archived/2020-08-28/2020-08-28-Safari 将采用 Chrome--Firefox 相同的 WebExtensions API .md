[Safari 将采用 Chrome/Firefox 相同的 WebExtensions API](https://www.solidot.org/story?sid=65376)
------

<div>
苹果<a href="https://developer.apple.com/news/?id=kuswih5l">宣布</a>，Safari 浏览器将采用  Chrome、Firefox 和 Edge 使用的相同扩展技术 WebExtensions API。Chrome 有着庞大的扩展生态系统，而随着所有主流浏览器都采用 WebExtensions API，这一扩展技术已成为事实上的标准，这也方便用户从一个浏览器切换到另一个浏览器。Firefox 是从 v57 开始正式采用 WebExtensions API 的。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
