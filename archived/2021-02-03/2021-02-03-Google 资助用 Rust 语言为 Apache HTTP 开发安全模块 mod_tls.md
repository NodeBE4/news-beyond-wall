[Google 资助用 Rust 语言为 Apache HTTP 开发安全模块 mod_tls](https://www.solidot.org/story?sid=66849)
------

<div>
Google 资助了 Internet Security Research Group(ISRG)的一个项目：用 Rust 语言为 Apache HTTP web server 项目<a href="https://www.zdnet.com/article/google-funds-project-to-secure-apache-web-server-project-with-new-rust-component/" target="_blank">开发</a>安全模块 mod_tls。在 Apache web server 中，<a href="https://httpd.apache.org/docs/current/mod/mod_ssl.html">mod_ssl</a> 用于支持建立 HTTPS 连接所需的加密操作。它是用 C 语言开发的，新的模块 mod_tls 将是用 Rust 语言开发，领导该项目开发的是软件咨询公司 Greenbytes 的创始人和 Apache HTTP Server 开发者 <a href="https://eissing.org/" target="_blank" rel="noopener noreferrer nofollow" data-component="externalLink">Stefan Eissing</a>。ISRG 希望，在完成开发之后 Apache HTTP web serve 团队将采用 mod_tls 作为默认模块取代年代悠久不安全的 mod_ssl。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
