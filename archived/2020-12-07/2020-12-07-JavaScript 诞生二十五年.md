[JavaScript 诞生二十五年](https://www.solidot.org/story?sid=66292)
------

<div>
编程语言 JavaScript <a href="https://www.zdnet.com/article/javascript-at-25-the-programming-language-that-makes-the-world-go-round/" target="_blank">诞生二十五年</a>，虽然<a href="https://developers.slashdot.org/story/20/12/04/2139202/javascript-turns-25">饱受批评</a>，但 JavaScript 已成为 Web 和浏览器应用的重要组成部分。Brendan Eich 在 1995 年花了 10 天设计出了 JavaScript 原型，在它大获成功之后，微软设计出了超集 TypeScript，之后 Google 的 V8 引擎，基于 V8 的 node.js，Facebook 的 React，等等帮助 JavaScript 迅速占领了从桌面到移动的广泛平台。JavaScript 现在是世界最流行的编程语言。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
