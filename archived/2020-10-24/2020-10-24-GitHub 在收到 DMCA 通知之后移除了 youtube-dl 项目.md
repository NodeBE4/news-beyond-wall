[GitHub 在收到 DMCA 通知之后移除了 youtube-dl 项目](https://www.solidot.org/story?sid=65900)
------

<div>
GitHub 收到了美国唱片业协会（RIAA）的 <a href="https://github.com/github/dmca/blob/master/2020/10/2020-10-23-RIAA.md" target="_blank">DMCA 通知</a>，声称  <a href="https://github.com/ytdl-org/youtube-dl" target="_blank">youtube-dl</a> 项目 (i) 规避 YouTube 等授权流媒体服务的技术保护措施；(ii)未经授权复制和传播 RIAA 会员公司的音乐视频和录音。在收到通知之后 GitHub 移除了 youtube-dl 项目。这一事件<a href="https://news.ycombinator.com/item?id=24872911">引发了争议</a>，因为 RIAA 的指控<a href="https://news.slashdot.org/story/20/10/23/2341202/youtube-dl-downloading-software-removed-from-github-by-riaa-takedown-notice">并不真实</a>。youtube-dl 是一个非常受欢迎的视频下载工具，允许用户下载 YouTube 和 Vimeo 等流媒体网站的视频，但并没有尝试规避 DRM 保护。youtube-dl 项目的一个问题是它使用了版权音乐作为下载示例。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
