[微软发布 Edge for Linux 的预览版本](https://www.solidot.org/story?sid=65870)
------

<div>
微软<a href="https://blogs.windows.com/msedgedev/2020/10/20/microsoft-edge-dev-linux/">正式发布了</a>基于 Chromium 的 Edge for Linux 预览版本，感兴趣的 Linux 开发者可以直接到 <a href="https://www.microsoftedgeinsider.com/zh-cn/download/">Edge Insider 网站</a>（Dev 渠道）下载 .deb 或 .rpm 包。Edge for Linux 目前支持 Ubuntu、Debian、Fedora 和 openSUSE 发行版。Dev 渠道中的版本每周更新一次。微软称，技术预览版本主要针对想要构建和测试网站和应用在 Linux 上表现的开发者。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
