[GitHub 是中国少数能自由发表言论的地方](https://www.solidot.org/story?sid=65516)
------

<div>
连线网站的一篇文章<a href="https://www.wired.com/story/china-github-free-speech-covid-information/">指出</a>，GitHub 是中国少数能自由发表言论的地方。Facebook 和 Twitter 等外国社交平台都被屏蔽了，微博和微信等国内社交平台则受到了严密监视。GitHub 这个代码托管平台被称为是中国最后一块言论自由的土地。因为 GitHub 使用 HTTPS ，中国无法屏蔽个别项目。但又不能屏蔽整个 GitHub 网站，因为它对中国的科技行业几乎是无价之宝。中国的开发者群体高度依赖于围绕 GitHub 的开源社区。屏蔽 GitHub 的代价太昂贵了，中国并非没有尝试过，但屏蔽之后科技行业的知名领袖如李开复争论称，这将会导致中国的程序员脱轨，失去竞争力和前瞻性。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
