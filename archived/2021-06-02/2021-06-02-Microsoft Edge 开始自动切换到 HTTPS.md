[Microsoft Edge 开始自动切换到 HTTPS](https://www.solidot.org/story?sid=67937)
------

<div>
如果网站支持 HTTPS，Microsoft Edge 通过 HTTP 访问该网站时<a href="https://www.bleepingcomputer.com/news/security/microsoft-adds-automatic-https-in-edge-for-secure-browsing/" target="_blank">将自动切换到 HTTPS</a>。该功能已包含在测试版本中，正在推送给部分挑选的 Microsoft Edge 92 用户。自动切换到 HTTPS 连接将能阻止中间人攻击，防止流量纂改。如果 Microsoft Edge 用户现在就想启用该功能，可以打开  edge://settings/privacy，启用 Automatically switch to more secure connections with Automatic HTTPS。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
