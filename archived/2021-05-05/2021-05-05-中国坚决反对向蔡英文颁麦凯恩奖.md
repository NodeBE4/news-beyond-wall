[中国坚决反对向蔡英文颁麦凯恩奖](https://www.zaobao.com/realtime/china/story20210505-1144422)
------

<div>
<p>中国驻加拿大使馆发言人今天（5日）称，哈利法克斯国际安全论坛（Halifax International Security Forum，HFX）不顾中方严正立场，执意“向中国台湾地区领导人蔡英文颁奖，严重违反一个中国原则，向‘台独’分裂势力发出错误信号，中方对此表示坚决反对。”</p><p>综合中国政府网与《自由时报》报道，哈利法克斯国际安全论坛（Halifax International Security Forum，HFX）昨天（4日）宣布，正式将今年的麦凯恩奖（John McCain Prize）颁赠给台湾总统蔡英文。颁奖影片介绍蔡英文是台湾首位女总统，也是华语世界最有权势的女性政治人物，并称中国大陆持续威胁台湾，而蔡英文在2020年连任，是台湾社会向大陆传达的清楚讯息，台湾无惧大陆。</p><p>中国驻加拿大使馆发言人指出，世界上只有一个中国，台湾是中国领土不可分割的一部分，中华人民共和国政府是代表全中国的唯一合法政府。一个中国原则是公认的国际关系准则和国际社会普遍共识，也是中加关系的政治基础。任何炮制“一中一台”、“两个中国”的图谋都将注定失败，任何损害中国国家主权和领土完整的行径都必将遭到痛击。</p><section id="imu"><div id="dfp-ad-imu1">        </div></section><p>发言人说，中方敦促哈利法克斯国际安全论坛充分认清台湾问题的高度敏感性，切实恪守一个中国原则，立即取消向蔡英文颁奖的错误决定，慎重妥善处理台湾问题，不要再向“台独”势力发出任何错误信号。</p><p> </p>      <div id="innity-in-post"></div><div id="dfp-ad-midarticlespecial">        </div>      <a href="https://subscribe.sph.com.sg/promotions/chinese-3-months-99-cents-promotion/?utm_campaign=cn3in199c&utm_medium=sph-publication&utm_source=zb&utm_content=subscribelink-free-endofarticle" target="_blank"><img class="view-desktop lazyload  " data-src="https://static.zaobao.com/s3fs-public/freemium_images/20210209/399-default-desktop-l_0_1.jpg" alt style="width:100%;" pinger-seen="true" src="https://static.zaobao.com/s3fs-public/freemium_images/20210209/399-default-desktop-l_0_1.jpg" referrerpolicy="no-referrer"></a>      <a href="https://subscribe.sph.com.sg/promotions/chinese-3-months-99-cents-promotion/?utm_campaign=cn3in199c&utm_medium=sph-publication&utm_source=zb&utm_content=subscribelink-free-endofarticle" target="_blank"><img class="view-mobile lazyload " data-src="https://static.zaobao.com/s3fs-public/freemium_images/20210209/399-default-mobile_1.jpg" alt pinger-seen="true" src="https://static.zaobao.com/s3fs-public/freemium_images/20210209/399-default-mobile_1.jpg" referrerpolicy="no-referrer"></a>
