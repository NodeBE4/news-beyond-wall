[微软宣布 Windows 11](https://www.solidot.org/story?sid=68121)
------

<div>
微软<a href="https://www.theverge.com/2021/6/24/22546791/microsoft-windows-11-announcement-features-updates">正式宣布了</a>它下一代操作系统 Windows 11。它与 Windows 10 的关系更像是新瓶旧酒。Windows 11 最大的变化是它的用户界面，包括新的开始菜单，更新了开始按钮，新 UI 与已经取消的  Windows 10X 相似，任务栏默认在底部中心位置显示。开始按钮抛弃了从 Windows 8 起引入的动态磁贴。微软称，Windows updates 将更小更有效率，它将在后台工作。 Windows 11 还直接整合了 Microsoft Teams 和 Xbox Game Pass。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
