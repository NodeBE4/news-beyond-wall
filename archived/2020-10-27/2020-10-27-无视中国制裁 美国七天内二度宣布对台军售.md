台湾国防部发言人史顺文27日证实，美国在一周内，第二次宣布对台总额约23.7亿美元的军售：“美国政府是在美东时间10月26号就‘鱼叉飞弹海岸防卫系统’的对台军售案，进行‘知会国会’的程序，顺利的话就可以在1个月后正式生效。”</p><p> </p><ul><li><b><a class="external-link" 台湾的在野党、国民党籍立委吕玉玲接受自由亚洲电台采访，也认同美方对台军售，像美国通过台湾旅行法，也是对《台湾关系法》有再升级保护的策略。面对这样紧张的气氛，美方给我们适时增加武力装备也是有需要的，大家都在军备角力，我们也要展现我们的肌肉。<br/> <br/> 美方26日宣布的100套鱼叉飞弹岸置防御系统军售案，包括400枚RGM-84L-4鱼叉II型地面发射飞弹、4枚RTM-84L-4鱼叉II型训练飞弹、411只集装箱、100辆鱼叉飞弹岸置防御系统机动发射器运输载具（发射车）、以及25辆雷达车等。<br/> <br/> 揭仲分析，中共如果要对台湾进行登陆作战，部队会从大登陆舰换到小登陆艇，抢滩上岸，大船换小船“换乘区”离台湾陆地不超过三十公里。但到了2025年，大陆可能用直升机或高速气垫船载运部队。随着中国解放军登陆战术、工具的改变，2025到2035年期间，所谓换乘区可能离台湾本岛更远，约六十、七十公里的时候，陆军部队火炮、国造雷霆2000无法打击2025年时的换乘区，射程不够远。<br/> <br/> 揭仲说，美国防部跟台湾国防部未来可能推算中共可能有的登陆载具。“我们希望达到多少的击杀性，根据这系列参数推演2025到2035，应付中共威胁需要买多少这种可以做远距精准攻击的弹药，包括多管火箭，目的是希望我们的攻击范围，可以从中国大陆的陆地一直到日后距台湾本岛60到70公里的换乘区。当中国解放军在沿海各省开始做部队物资集结时，我们就可以开始攻击船团出发、海上换乘区，进行重重打击火力。”<br/> <br/> 揭仲说，台湾虽然有本土自行研发的雄二飞弹、雄三飞弹，但产能可能跟不上，美方售台400枚鱼叉飞弹，可对解放军船舰发动大量飞弹攻击，以多层次火力打击消耗他们的攻击部队。<br/> <br/> 揭仲说：“国军就可运用各式各样飞弹、远距火箭、攻击性飞弹，从军舰、飞机、陆地发射， 到时我们能面对来自不同方向、使用不同飞弹型号的攻击，增加中国大陆保护攻击部队的困难度。”</p><p> </p><p><b><div class="image-inline captioned" style="width:640px;"><div style="width:640px;"><img alt="中国大陆外交部发言人汪文斌27日表示，美国对台军售严重违反“一个中国”原则。（路透社）" 中方制裁影响不大</b></p><p>在此之前，中国在26号宣布制裁参与对台军售的美国企业和个人。中国外交部发言人赵立坚26日针对美国21日对台约18亿美元的军售进行了反击：“为了维护国家利益，中央决定采取必要措施，对参与此次对台军售的洛克西德马丁、波音防务、雷神等美国企业，以及在售台武器过程中，表现恶劣的美国有关个人和实体，实施制裁。”<br/> <br/> 对于中国外交部宣布制裁参与美方军售的军火商和所谓“恶劣”个人与实体，揭仲说，这三家军火商主要客户是美国国防部，中国市场对他们如“九牛一毛”，中方宣布制裁的波音公司指的是“波音防务”，也是武器制造部门，所以中方扬言的制裁应没有多大影响力。<br/> <br/> 揭仲说：“美制的武器系统，到目前为止是不能销中国大陆的，所以他实际能制裁的项目很有限，这些公司相较于他们在武器生产上的获利，在中国大陆有一些投资计划，有一些小的合作计划是可以被牺牲掉的。”<br/> <br/> 此外，台湾国防部26日表示，解放军一架运8反潜机、一架运8技术侦察机侵犯台湾西南防空识别区（ADIZ），空军除派遣空中巡逻兵力应对，也实施广播驱离、防空飞弹追踪监控；这是自9月16日以来，第24度扰台。<br/> <br/> 北台湾小镇淡水27日则出现大批坦克装甲车移防，引发民众热议。国防部表示，为战备周操演训练，辖内陆海空进行实兵实装实作，借由地形、地物、桥梁、涵洞、民间厂房及地下设施等坚固的建筑物，运用这些伪装达到防护的目的。<br/> <br/> <br/> 自由亚洲电台记者 夏小华  台北报导 责编 许书婷  申铧  网编 瑞哲</p>
