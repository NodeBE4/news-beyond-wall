[阿里巴巴开源基于 PostgreSQL 的数据库 PolarDB](https://www.solidot.org/story?sid=67902)
------

<div>
阿里巴巴<a href="https://github.com/alibaba/PolarDB-for-PostgreSQL">开源</a>了它基于 PostgreSQL 的数据库 PolarDB，源代码采用 Apache Version 2.0 License 托管在 GitHub 上。阿里巴巴关联公司蚂蚁集团<a href="https://finance.sina.com.cn/tech/2021-05-27/doc-ikmyaawc7861697.shtml">据报道</a>将很快开源另一个数据库 OceanBase。PolarDB 扩展了 PostgreSQL，支持全局数据一致性、ACID 跨数据库节点、分布式 SQL 处理、数据冗余和高可用性，设计在引入新功能的同时维持兼容性。<a href="https://www.alibabacloud.com/zh/product/oceanbase" target="_blank">OceanBase</a> 去年以公司模式独立运营，兼容 MySQL 以及 Oracle，数据分析基准测试（TPC-H）排名第一。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
