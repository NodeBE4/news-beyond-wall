[Epic 收购 RAD Game Tools](https://www.solidot.org/story?sid=66600)
------

<div>
Epic Games <a href="https://www.epicgames.com/site/en-US/news/epic-acquires-rad-game-tools?sessionInvalidated=true">宣布</a>收购游戏软件开发公司 RAD Game Tools。RAD 的 Bink Video 视频格式因其低开销和快速解压缩而被游戏公司广泛采用，几乎所有 3A 游戏都使用Bink Video。Epic 称，RAD Game Tools 的产品和技术被用在近 2.5 万款游戏中，被包括它在内的世界顶尖游戏公司使用，它计划将 RAD 的技术整合到虚幻引擎中，让开发者社区和玩家共同受益。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
