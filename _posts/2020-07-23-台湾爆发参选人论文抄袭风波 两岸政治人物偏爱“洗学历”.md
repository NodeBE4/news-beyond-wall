[台湾爆发参选人论文抄袭风波 两岸政治人物偏爱“洗学历”](https://www.rfa.org/mandarin/yataibaodao/gangtai/hcm1-07232020081554.html)
------

<div>
<p> </p><p>台湾的国民党高雄市长候选人李眉蓁爆发硕士论文涉嫌抄袭整本123页，李眉蓁本人22日发表声明，将放弃在选举公报学历栏中删除中山大学在职硕士班学位。为何两岸不少政治人物为了漂亮学历，都要在步入政坛后还拼命“洗学历”？<br/> <br/> 李眉蓁先是被发现，论文有4页抄袭现任侨委会委员长童振源教授所发表的文章“两岸政经互动”，第二节“分析架构”。紧接着台媒镜周刊又披露，李眉蓁论文全文，扣掉前后的摘要、目录及参考文献，本文共123页，有高达96%的页数都涉嫌抄袭自台北大学公共行政暨政策学系硕士生雷政儒在2000年7月出版的论文《两岸经贸互动与台商投资之演变分析》，同样未注明出处。</p><p> </p><ul><li><b><a 李眉蓁：“我在此郑重宣布：放弃我中山大学在职硕士班学位也会向高雄市选举委员会申请，选举公报学历栏内，删除这项注记，期盼大家回归市长补选的政见主轴。”<br/> <br/> 李眉蓁的声明当中，并未承认“抄袭”的部分，仅表示，“因个人的不察，造成学术伦理的争议，我应该表示歉意，并展现负责的态度。”<br/> <br/> 而李眉蓁攻读硕士的国立中山大学23日也发表声明，有关李眉蓁学术伦理案，学校已在7月20日启动学术伦理审查机制，社会科学院已成立审定委员会，秉持公平、公正、客观、明快原则，送校外专业领域公正学者审查；若事证明确，会加快处理、缩短期程，并坚守程序正义。若审定委员会认定违反学术伦理，最重将撤销学位；若指导教授失职，将移送教评会检讨。<br/> <br/> <b>学历成为参政基本门槛 政治人物“洗学历”毒药加春药</b><br/> <br/> 国立台湾师范大学政治学研究所教授曲兆祥告诉本台，台湾过去一、二十年的时间，高等教育发展得很快，在大量制造下，确实出现良莠不齐的现象。很多学位出现很多瑕疵，所以这几年高教单位基本上都蛮重视“学术伦理”，抄袭的问题，李眉蓁的事情不是唯一的个案。</p><p> </p><p><div class="image-inline captioned" 曲兆祥：“这东方文化与西方文化的差别，因为东方人特别是华人社会，还是有“学而优则仕”这种观念，你要“仕”要“学而优”嘛！ 这是一个文化传承，虽然现在大家不一定都这么想，但是“学历”还是被大家重视。”<br/> <br/> 台湾大学政治系副教授彭锦鹏接受本台访问时直白地说，对台湾政治人物而言，“洗学历”就像“春药加毒药”！“选民的脑筋、价值观很难改变，因为学历最简单的是大家相信一种价值叫“公平、公正、公开”，你有学历多少可以表示你有什么样的能力，不管这个学历是真的假的有没有用，大家都相信。”<br/> <br/> 彭锦鹏直指，台湾政治人物“洗学历”现象背后是因为代价太便宜。因为付出的钱实在是太低了，以至于能力够的人愿意花钱。他反问，大家敢随便到美国去拿个硕士、不是吗？不敢，因为要花很多钱。<br/> <br/> 彭锦鹏：“我们相对来讲非常廉价，(念个国立大学在职班要新台币100万吗？)不用，6万乘以四学期，25到30万就可以。论文枪手也不会太多，台币1-20万就有。”</p><p> </p><p><div <br/> 毛泽东前秘书李锐过去曾在专访中指出，中共总书记习近平仅有“小学程度”，著名评论员高新曾在专栏评论习近平的清华大学博士学经历备受质疑。<br/> <br/> 对于两岸的政治人物不约而同都有“高学历”迷思，曲兆祥认为，东方人特别是华人社会，还是有“学而优则仕”观念，这是一个文化传承，虽然现在大家不一定都这么想，但是“学历”还是被大家重视。<br/> <br/> 曲兆祥：“华人社会在乎学历无可厚非，但是为了学历，搞虚假、搞坑拐骗，这就真的问题比较大。两岸都有一种行业叫做论文代写，过去有一段时间大陆连论文都不必写，直接作假的毕业证书。”<br/> <br/> 英国前首相梅杰仅高中毕业，难道两岸不能接受一个非硕博士的领导人物吗？彭锦鹏说，可以，但是这个人必须要“非常杰出、要把能力表现出来，现代的社会越来越困难。”现代的社会认为，如果没有学历，那就是视野、知识有问题。<br/> <br/> 彭锦鹏：“对岸跟中国传统一模一样，他们更严重，他们买得更严重。(在对岸这不是新闻？)不是新闻，他们可以抄整本，名字、标题改一下，就算是自然科学也可以把所有数据引用，换另一个title(名目)就可以。”<br/> <br/> 曲兆祥说，台湾的大学亡羊补牢的做法是，要求所有的学生在口试论文之前，一定要交一份在其他论文比对过相似度的应用软体证明书，相似度超过一定比例，学校就不会接受。彭锦鹏说，对岸目前似乎还没有相对机制，因为学校实在太多了。<br/> <br/> <br/> 自由亚洲电台记者黄春梅 台北报道 责编 许书婷 胡力汉 网编 瑞哲</p>
