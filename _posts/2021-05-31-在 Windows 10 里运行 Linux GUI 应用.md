[在 Windows 10 里运行 Linux GUI 应用](https://www.solidot.org/story?sid=67915)
------

<div>
微软已经释出了 Windows 10 的预览版本，允许用户直接运行 Linux GUI 应用。那么<a href="https://www.bleepingcomputer.com/news/microsoft/hands-on-with-wslg-running-linux-gui-apps-in-windows-10/" target="_blank">如何在 Windows 10 里运行 Linux GUI 应用</a>？首先支持 Windows Subsystem for Linux GUI(WSLg)需要 Windows 10 Insider build 21364 或更新版本，然后安装 Windows Subsystem for Linux (WSL) 和 Linux 发行版如 Ubuntu。现有的 WSL 用户可以通过更新加入对 WSLg 的支持，一旦更新之后用户还需要确保安装的 Linux 发行版是否配置使用 WSL 2，如果发行版配置使用的是 WSL 1，那么需要通过命令更新配置使用 WSL 2。之后你就可以通过  WSLg 运行 Linux GUI 应用了。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
