[Facebook 成为 Linux 基金会的白金会员](https://www.solidot.org/story?sid=65248)
------

<div>
社交巨人 Facebook <a href="https://www.linuxfoundation.org/blog/2020/08/facebooks-long-history-of-open-source-investments-deepens-with-platinum-level-linux-foundation-membership/" target="_blank">加入了 Linux 基金会</a>，成为白金会员。作为白金会员，Facebook 开源负责人  Kathy Kam 将加入基金会的董事会。Facebook 向很多开源项目贡献了代码，如 React JS项目，它也是 Linux 基金会托管的项目  <a href="https://prestodb.io/" target="_blank" rel="noopener noreferrer nofollow" data-component="externalLink">Presto</a>、<a href="https://graphql.org/" target="_blank" rel="noopener noreferrer nofollow" data-component="externalLink">GraphQL</a>、<a href="https://osquery.io/" target="_blank" rel="noopener noreferrer nofollow" data-component="externalLink">Osquery</a> 和 <a href="https://onnx.ai/" target="_blank" rel="noopener noreferrer nofollow" data-component="externalLink">ONNX&nbsp; </a>的主要贡献者。其它基金会的白金会员包括：Google、华为、IBM、英特尔、微软、三星和腾讯等。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
