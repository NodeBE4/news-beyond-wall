[Windows 计算器移植到到 Linux](https://www.solidot.org/story?sid=65830)
------

<div>
微软去年公开了计算器应用的源代码，<a href="https://snapcraft.io/blog/the-windows-calculator-on-linux-with-uno-platform">现在</a>  Uno Platform 社区成员将其<a href="https://platform.uno/blog/windows-calculator-on-linux-via-uno-platform">移植</a>至到 Linux 平台，用 Snap 格式打包发布在 Canonical 的 <a href="https://snapcraft.io/uno-calculator" target="_blank">Snapstore</a>（如果使用 Ubuntu 系统可在终端里输入 snap install uno-calculator）。Snap 是 Canonical 开发的应用打包格式，旨在减少依赖库问题，但最近因为 Canonical 在 Ubuntu 操作系统中强推 Snap 应用而在开源社区引发了争议。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
