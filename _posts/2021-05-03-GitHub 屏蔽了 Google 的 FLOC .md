[GitHub 屏蔽了 Google 的 FLOC](https://www.solidot.org/story?sid=67677)
------

<div>
GitHub<a href="https://tech.slashdot.org/story/21/05/02/1556241/github-joins-movement-against-googles-floc" target="_blank"> 加入了</a>反对 Google 的第三方 cookie 跟踪替代技术 FLOC（Federated Learning of Cohorts）的阵营。它的网页服务 GitHub Pages <a href="http://github.blog/changelog/2021-04-27-github-pages-permissions-policy-interest-cohort-header-added-to-all-pages-sites/" target="_blank">屏蔽了 FLOC</a>，来自 github.io 域名的 GitHub Pages 页面使用了 Permissions-Policy: interest-cohort=() 报头设置，使用定制域名的页面设置没有改变。在这之前，最流行的内容管理平台 WordPress 宣布它计划在其平台上默认自动屏蔽 FLOC。WordPress 还表示它考虑加入设置允许管理员控制是否允许 FLOC。Firefox、Brave 和 Vivaldi 也都采取了类似的行动。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
