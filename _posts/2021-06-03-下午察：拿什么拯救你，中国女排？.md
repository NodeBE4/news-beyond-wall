<p>继前天2比3负于加拿大后，中国女排在昨天举行的2021年世界女排联赛预赛第六轮比赛中，再以0比3惨败土耳其队。</p><p>在被誉为“奥运热身赛”的世联赛前两阶段比赛中，前奥运冠军中国队以三胜三负的成绩排名第八，这不仅让主教练郎平大动肝火，也让其世界排名第一的位置被美国取代，更让许多球迷为女排不到两个月后的奥运之旅捏一把冷汗。</p><p><strong>“郎平眼睛肿了”上热搜</strong></p><section id="imu"><div id="dfp-ad-imu1">        </div></section><p>众所周知，本次世界女排联赛上郎平带的多为替补阵容，这是她的策略：通过高水平的对抗来锻炼替补，并以此确定东京奥运会名单。</p><p>因此，上周（5月26日）以替补队员为主的中国女排在比赛中输给全主力阵容的日本时，仍得到理解，但前天以2比3不敌加拿大队就没那么让人淡定了。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/jianada_xinhua.jpg?lk_9U2om7h.IxEYlSq7Rrttbj7d5oGrz&itok=1qodo-Dv" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>这支加拿大球队有不少是业余和学生球员，本届比赛的成绩是“一场不胜”，岂料，这项记录竟然因中国队终结了。（新华社）</figcaption></figure>                  <p>要知道，这支加拿大球队有不少是业余和学生球员，本届比赛的成绩是“一场不胜”，岂料，这项记录竟然因中国队终结了，中国女排为这支业余球队实现了“零”的突破。这甚至是加拿大女排历史上第一次战胜中国女排，无怪乎加拿大教练激动得泪洒球场。</p><p>赛后，同样洒泪的，可能还有中国队主教练郎平？她在隔天对阵世界劲旅土耳其队时一直安静的呆坐着，戴着口罩的她露出一双疲惫红肿的双眼，被媒体指“像是大哭一场后的样子”。</p><div id="innity-in-post"></div><div id="dfp-ad-midarticlespecial">        </div><p>甚至在比赛暂停阶段，郎平也没有站起来进行指导，而是由担任副手的安家杰代替。随后“郎平眼睛肿了”冲上微博热搜。</p><p>这与她在前一场中激动得大骂甚至快哭出来的状态大相径庭。对郎平来说，即使不是全主力出战，即使是为了筛选、锻炼二线队员，“中国女排”破天荒地输给一支业余队也仍在她意料之外。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/langping_yanzhong_1_jiequ.jpg?43iMRLmaQlBdu.Pwl26T1NZtFnmaqzp1&itok=fXQwBSol" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>她在隔天对阵世界劲旅土耳其队时一直安静的呆坐着，戴着口罩的她露出一双疲惫红肿的双眼，被媒体指“像是大哭一场后的样子”。（视频截取）</figcaption></figure>                  <p>犹记得，郎平在2005年的大冠军杯比赛中曾带领一只由公司职员和大学生等组成的纯业余美国女排队，以3比0击败了当时的雅典奥运冠军中国队，并最终取得亚军名次。</p><p>时移世易，同样是业余队击败专业队的戏码，但角色却换位了。可以想象郎平当时有多扎心，</p><p><strong>中国女排实力几何？</strong></p><p>以替补阵容来评判中国女排的整体实力肯定不准确，但二线队员太弱，板凳太浅，主力和替补之间差距太大，显然是郎平这个主教练和中国女排不可回避的现实。</p><p>以对阵土耳其为例，两队上一次交手是在2019年的奥运资格赛上，彼时全主力阵容的中国女排是以3比0获得大胜，这可以从侧面证明中国女排主力阵容与替补间的巨大差距。</p><p>何况，土耳其女排在本次大赛中排出的阵容也有保留，并不是征战奥运会的完整阵容，但仍能基本表现出正常水平，并获得五战全胜的理想成绩，更反衬出中国替补队员的弱势。</p><p>去年末世界排联公布的2020年最后一期世界排名中，中国女排年终排名第一。当时与排名第二的美国女排积分差距只有9分。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/screenshot_1.jpg?PH2_cMlH6boP6oAy9Bs5H2soWEglkqe6&itok=3q5wCby6" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>中国和美国排名积分差距拉大到49分，与巴西的差距缩小仅有三分。（世界排联官网）</figcaption></figure>                  <p>随着中国女排在世联赛前两阶段比赛中输掉三场，其世界排名被美国反超后还被进一步拉大差距，而排在中国身后的巴西女排也步步紧逼。</p><p>更糟糕的是，以目前在世联赛的排名和对阵来看，中国女排之后即便让朱婷这样的明星球员回归，冲击前四名的希望也相当渺茫，而联赛中的最终排位也会直接影响其世界排名，中国第二名的位置在大赛落幕后还能否坐稳，得划个问号。</p><p><strong>奥运冠军滤镜恐被打破</strong></p><p>作为上届奥运冠军，中国对即将出征东京的女排队伍寄予厚望。但除了替补队员差距巨大外，有资深球迷指出，上届奥运会之后，包括女排前队长、二传魏秋月、主攻惠若琪以及副攻徐云丽等老队员都退役了，而这些位置上并没有找到和她们实力完全相当的接替者。</p> src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/zhangchangning_xinhua.jpg?_74SkHhyjxHHHO35tYQPrgktbXSuowsw&itok=SGCuQ_EJ" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>以作为为数不多主力出现在对阵加拿大比赛中的主攻张常宁为例，虽然她在比赛中力挽狂澜独拿22分，但也出现了发球不过网等致命失误。（新华社）</figcaption></figure>                  <p>以作为为数不多主力出现在对阵加拿大比赛中的主攻张常宁为例，虽然她在比赛中力挽狂澜独拿22分，但也出现了发球不过网等致命失误。</p><p>另外，近几年争议较大的“自由人”问题在这次比赛中更加突出，在2016年里约奥运会最佳自由人林莉状态迅速下滑后，王梦洁成为中国队的首发自由人，也因此跻身女排“七仙女”之列，但她也是此次大赛引起争议最多的队员。</p><p>有报道指出，王梦洁在本次比赛中不但在一传环节的发挥没有太多亮点，就连其赖以成名的防守，也没有展现出正常水准，以至于“王梦洁状态”频频冲上热搜，单是世联赛前五轮，她就上了三次热搜。</p><p>有球迷此前就做过相关分析，指出虽然获得了2019年女排世界杯最佳自由人，但实际上王梦洁与意大利德吉纳罗、巴西女排莱娅、多米尼加卡斯蒂略等世界顶级自由人有一定差距，因此中国女排在自由人这个位置只能算是世界二流水平。</p><p>也就是说，连主力阵容也未必可与上届奥运阵容旗鼓相当，更不要提被加拿大业余队干掉的替补队员了。</p><p>有球迷甚至说，要接受中国女排青黄不接的事实，单靠一个朱婷，是没有办法赢得奥运冠军的。</p><p><strong>拿什么拯救你，中国女排？</strong></p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/zhuting_cnsz.jpg?AaBhRsBWGBiFxqtrTUVbuh99uNCemIC2&itok=Z4XYd_c2" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>有球迷甚至说，要接受中国女排青黄不接的事实，单靠一个朱婷，是没有办法赢得奥运冠军的。（中新社）</figcaption></figure>                  <p>如果一个朱婷无法让女排夺冠，那么一个郎平可能拯救整个女排吗？</p><p>郎平对于女排队伍青黄不接的压力早有感知，而且她也很清楚，里约夺冠有一部分原因是对手对当时作为秘密武器的新人朱婷不熟悉。</p><p>郎平在今年3月接受央视采访时直言，这届奥运的核心问题就是，她必须找到一个人站出来分担朱婷在进攻环节的压力，因为朱婷一定会被对手重点“照顾”。</p><p>但目前可供选择的队员中，刘晓彤、张常宁、刘晏含三名老将的技术特点对于中国女排对手来说不仅已经接近透明，而且实力也没有明显提升。其他年轻队员又有大赛经验不足的问题。</p><p>换言之，仍然是板凳深度不够，队员青黄不接的问题。这个问题不解决，不仅可能无缘卫冕奥运冠军，还会阻碍中国女排下一个周期的发展。比赛本来就是有赢有输，对一支球队长胜不败的期望并不现实，但球队正常发挥该有的水平却是合理期待。久经沙场的郎平有着钢铁般的意志，而今距离下一场硬仗只剩下约2个月时间，她有绝招扭转劣势吗？东京奥运见分晓。</p>
