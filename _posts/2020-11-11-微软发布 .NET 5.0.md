[微软发布 .NET 5.0](https://www.solidot.org/story?sid=66063)
------

<div>
微软<a href="https://devblogs.microsoft.com/dotnet/announcing-net-5-0/">发布了 .NET 5.0</a>。新版包括了 <a href="https://devblogs.microsoft.com/aspnet/announcing-asp-net-core-in-net-5/">ASP.NET Core</a>，<a href="https://devblogs.microsoft.com/dotnet/announcing-the-release-of-ef-core-5-0/">EF Core</a>，<a href="https://devblogs.microsoft.com/dotnet/c-9-0-on-the-record/">C# 9</a> 和 <a href="https://devblogs.microsoft.com/dotnet/announcing-f-5/">F# 5</a>，.NET 库增强了 Json 序列化、正则表达式和 HTTP（HTTP 1.1、HTTP/2）的性能。Visual Studio 用户需要 Visual Studio 16.8 以上版本才能使用 .NET 5.0。 .NET 5.0 是微软统一 .NET 堆栈和生态系统的首个版本，它计划在每年的 11 月发布一个新版本，这意味着 2021 年的 11 月将会发布&nbsp; .NET 6.0。.NET 5.0 不是长期支持版，它将支持到 .NET 6.0 发布 3 个月之后即 2022 年 2 月中旬，.NET 6.0 是长期支持版本，将支持三年。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
