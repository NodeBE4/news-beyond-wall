[pip 20.3 释出](https://www.solidot.org/story?sid=66239)
------

<div>
Python Packaging Authority 宣布<a href="https://pyfound.blogspot.com/2020/11/pip-20-3-new-resolver.html">释出</a> pip 20.3。由于改变了默认的依赖解析器，这个版本可能会造成破坏。pip 项目已经释出了如何测试和迁移到新解析器的<a href="https://pip.pypa.io/en/latest/user_guide/#changes-to-the-pip-dependency-resolver-in-20-3-2020">用户指南</a>。除了新的解析器，其它变化包括 Python 3.5 将在 pip 21.0 中移除，不再建议使用；pip 21.0 将在 2021 年 1 月释出，这个版本也将停止支持 Python 2.7，不再支持 Python 2.x 系列。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
