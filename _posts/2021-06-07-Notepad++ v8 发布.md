[Notepad++ v8 发布](https://www.solidot.org/story?sid=67977)
------

<div>
文本编辑器 Notepad++ <a href="https://notepad-plus-plus.org/downloads/v8/">释出了 v8</a>。主要新特性包括：深色模式（Dark Mode）、ARM64 构建版本，工具栏 Fluent UI 图标，免干扰模式，深色模式 API NPPM_ADDTOOLBARICON_FORDARKMODE，一次点击操作关闭所有标签，移除 Microsoft Bing 搜索，多个 bug 修正等等。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
