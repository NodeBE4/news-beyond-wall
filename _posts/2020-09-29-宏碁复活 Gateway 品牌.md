[宏碁复活 Gateway 品牌](https://www.solidot.org/story?sid=65686)
------

<div>
<a href="https://en.wikipedia.org/wiki/Gateway,_Inc." target="_blank">Gateway</a> 曾经是 1990 年代非常知名的 PC 品牌，它成立于 1985 年，1999 年被 AOL 收购，2007 年被宏碁收购。现在，宏碁<a href="https://www.theverge.com/2020/9/16/21439568/gateway-walmart-acer-budget-laptops-amd-intel-colors-price-release-date">决定复活</a> Gateway 品牌，与沃尔玛合作由其独售一系列 Gateway 冠名的笔记本和平板电脑，主打中低端市场，笔记本电脑最低仅售 179 美元，最高也只售 999 美元。Ars <a href="https://arstechnica.com/gadgets/2020/09/battle-of-the-350-laptops-acer-swift-1-vs-gateway-ryzen-3-3200u/">测试了</a>宏碁的两款 350 美元价位的笔记本电脑，其一是 378 美元的 Acer Swift 1，配备了 Pentium Silver N5000 处理器，其二是 350 美元的 Gateway GWTN141-2，配备 Ryzen 3 3200U 处理器，结果不出人意料，配备 AMD CPU 的 Gateway 表现胜于英特尔处理器的 Swift 1。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
