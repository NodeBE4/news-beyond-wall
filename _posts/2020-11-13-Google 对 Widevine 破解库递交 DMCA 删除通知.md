[Google 对 Widevine 破解库递交 DMCA 删除通知](https://www.solidot.org/story?sid=66087)
------

<div>
Widevine 是 Google 开发的 DRM 技术，Widevine CDM（内容解密模块）和 Widevine License Server 被用于流媒体内容的 DRM 保护，被 Disney+、Netflix、Amazon Prime Video、YouTube 和 Hulu 等流媒体服务商广泛使用。11 月 9 日 Google 以侵权为由向 GitHub <a href="https://github.com/github/dmca/blob/master/2020/11/2020-11-09-Google.md" target="_blank">递交 DMCA 删除通知</a>，要求 GitHub 删除数十个破解 Widevine  DRM 保护技术的库（都是同一个库的镜像）。GitHub 已经按照 Google 的要求移除了这些库。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
