[GitHub 为解雇犹太程序员道歉](https://www.solidot.org/story?sid=66695)
------

<div>
GitHub <a href="https://www.theverge.com/2021/1/17/22235913/github-significant-mistakes-were-made-firing-jewish-employee-nazis">为解雇一名犹太程序员道歉</a>，他称呼冲击美国国会的特朗普支持者为纳粹，警告同事注意安全。这名未公开身份的程序员是在内部的 Slack 聊天室里对华盛顿特区的同事发出这一警告的，称纳粹来了。他的警告被一名同事批评使用了制造分裂的修辞。他的企业账号不久被冻结，两天后他遭到解雇。这名犹太程序员有家人曾死于纳粹屠杀。他称公司人力资源在解雇时给出的理由是未指明的“行为模式”。有大约 200 名雇员在一封内部信函中署名抗议对他的解雇。GitHub 雇佣了一家独立法律事务所对此事展开调查，之后承认犯下了严重错误，公司人力总监 Carrie Olesen 也已经宣布辞职。GitHub 表示希望重新雇佣这名程序员。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
