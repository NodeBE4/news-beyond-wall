[微软称下一代 Windows 即将到来](https://www.solidot.org/story?sid=67870)
------

<div>
在微软开发者大会 Windows at Build 2021 上，CEO Satya Nadella <a href="https://tech.slashdot.org/story/21/05/25/1833203/microsoft-says-next-generation-of-windows-coming-soon" target="_blank">透露</a>下一代 Windows 即将到来，称这次更新是 Windows 过去十年最重要的更新之一。他没有透露更多细节，只是表示 Windows 10 已被超过 13 亿人使用。有流言称微软最早将在 6 月份举行一场 Windows 活动，暂时不清楚下一代 Windows 是否还是叫 Windows 10。自 2015 年以来，微软没有再改变 Windows 的版本号。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
