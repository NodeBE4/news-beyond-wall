[Java 1.0 发布 25 周年](https://www.solidot.org/story?sid=66747)
------

<div>
1996 年 1 月 23 日，Sun Microsystems <a href="https://www.infoq.com/news/2021/01/java-turns-25/" target="_blank">宣布发布 Java 1.0</a>。Sun 在 1995 年发布了 Java 的 alpha 版本，但 1.0 版本的发布标志着一段延续至今的旅程的开始。新兴的互联网将 Java 语言推向了大众，编译器和软件库的免费提供对当时也产生了巨大的冲击。GCC 这样的开源项目已经存在了十年，但绝大多数操作系统只有商业工具能构建交互式图形软件。Java 程序能编写和发布到互联网上，能在不同操作系统上运行这一事实在当时是独一无二的。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
