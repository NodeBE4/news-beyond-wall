[Linux Journal 再次恢复上线](https://www.solidot.org/story?sid=65619)
------

<div>
2017 年和 2019 年先后<a href="https://www.solidot.org/story?sid=61648" target="_blank">两次宣布</a>停止出版的 Linux 期刊 Linux Journal 再次<a href="https://www.linuxjournal.com/content/linux-journal-back">恢复上线</a>，这一次它成为了 Slashdot Media 的一部分。Linux Journal 表示将会尽快重新发布内容，正在寻找 Linux 新闻和 Linux 指南的撰稿人，社区和评论的版主。它暂时没有恢复订阅模式的计划，网站将发布免费内容。Slashdot Media 旗下包括了 Slashdot 网站和 SourceForge 源代码托管平台。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
