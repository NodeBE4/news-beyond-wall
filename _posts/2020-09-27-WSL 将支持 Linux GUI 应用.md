[WSL 将支持 Linux GUI 应用](https://www.solidot.org/story?sid=65664)
------

<div>
微软 WSL 项目的产品经理在 Twitter 上<a href="https://twitter.com/craigaloewen/status/1308452901266751488">演示了</a> WSL 对 Linux GUI 应用的支持。现有的 Windows Subsystem for Linux（WSL）子系统支持运行命令行 Linux 应用，通过 Linux GUI App Integration(WSLG)未来开发者将能运行有图形用户界面的 Linux 应用。微软此举旨在吸引开发者在 Windows 而不是 Linux 发行版上开发、运行和测试 Linux 应用程序。该公司的开发者在最近举行的 Linux Plumbers Conference <a href="https://linuxplumbersconf.org/event/9/contributions/611/attachments/702/1298/XDC2020_-_X11_and_Wayland_applications_in_WSL.pdf">介绍了相关细节</a>（PDF）。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
