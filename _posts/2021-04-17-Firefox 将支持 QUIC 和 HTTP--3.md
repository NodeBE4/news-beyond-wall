[Firefox 将支持 QUIC 和 HTTP/3](https://www.solidot.org/story?sid=67518)
------

<div>
Mozilla 宣布，<a href="https://www.mozilla.org/en-US/firefox/nightly/all/" target="_blank">Firefox Nightly</a> 和 <a href="https://www.mozilla.org/en-US/firefox/all/#product-desktop-beta" target="_blank">Firefox Beta</a> 默认启用了对 QUIC 和 HTTP/3 协议的支持，计划在五月底发布的正式版本默认将支持 HTTP/3。HTTP/3 是新版的 HTTP 协议，相比 HTTP/2 它有三个方面的性能改善：基于 UDP 连接时间较短；没有线端阻塞；能更好的检测和修复丢包。如果 Web server 支持 Firefox Nightly 和  Firefox Beta 将会自动尝试 HTTP/3。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
