[GitHub 警告重新发布 YouTube-DL 的用户](https://www.solidot.org/story?sid=65980)
------

<div>
上个月，GitHub 在收到美国唱片业协会（RIAA）的 DMCA 通知后<a href="https://www.solidot.org/story?sid=65900" target="_blank">移除了</a>非常受欢迎的 youtube-dl 项目，此举激怒了开发者和用户，他们通过重新将 youtube-dl 项目的源代码上传到 GitHub 来表达不满。GitHub 曾表示它也想帮助该项目重新上线，但现在它<a href="https://torrentfreak.com/github-warns-users-reposting-youtube-dl-they-could-be-banned-201102/">警告发布</a> youtube-dl 源代码的用户他们的账号可能会临时屏蔽，甚至可能封禁。    GitHub 法律顾问 Jesse Geraci 警告用户，他们重新发布因 DMCA 下架项目的源代码这一做法违反了 DMCA 政策和网址的服务条款，其后果可能包括删除内容并临时吊销账号。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
