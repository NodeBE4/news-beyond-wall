[微软移除了允许用户永久关闭 Windows Defender 的注册表项](https://www.solidot.org/story?sid=65324)
------

<div>
Windows 10 在最近的一次更新中<a href="https://www.pcgamer.com/permanently-disabling-windows-10s-built-in-virus-scanner-is-no-longer-an-option/">移除了</a>允许用户永久关闭 Windows Defender 的注册表项。Windows Defender 是微软内置在 Windows 10 操作系统中的杀毒软件，在这之前用户可以通过编辑一个注册表项 DisableAntiSpyware 永久关闭该应用。微软在其<a href="https://docs.microsoft.com/zh-cn/windows-hardware/customize/desktop/unattend/security-malware-windows-defender-disableantispyware?ranMID=42411">支持文档</a>中解释说，DisableAntiSpyware 设计被 OEM 和 IT 专业人士使用去禁用 Microsoft Defender Antivirus，部署另一种杀毒软件产品。它是一个旧的设置，不再需要，因为 Microsoft Defender 会在检测到另一种杀毒软件之后会自动禁用。该设置不是设计给消费者设备使用的，因此它决定移除该注册表项。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
