[一名雇员在伊朗登陆之后 GitHub 封锁了整个公司账号](https://www.solidot.org/story?sid=66567)
------

<div>
一家公司的雇员回伊朗看望父母，期间打开笔记本电脑登陆了 GitHub 账号，结果是整个公司的账号被封了。GitHub 是一家在美国注册的公司，为了遵守美国对伊朗的出口管制，包括 GitHub、Google Developer、AWS、Docker、NPM、Android、Go、Kubernetes 和 GitLab 等服务都屏蔽了伊朗人的访问。但这次公司账号被屏蔽不是因为伊朗的雇员，<a href="https://twitter.com/sebslomski/status/1344219609923276801">而是雇员去了伊朗</a>。GitHub 支持表示对于错误的屏蔽可以提起上诉。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
