<p>因宣称推翻爱因斯坦的相对论，中国舆论场这一两天来，李子丰的关注度比李子柒还高。</p><p>李子丰是谁？</p><p>河北燕山大学研究员。燕山大学官网显示：1962年出生，学历博士研究生，车辆与能源学院的研究员、教师，研究领域为油气井杆管柱力学、油气钻采工程。</p><section id="imu"><div id="dfp-ad-imu1">        </div></section><p>爱因斯坦相对论是关于时空和引力的理论，颠覆了人类对宇宙和空间的认识，是现代物理学的基石，是世界公认的科学经典理论。</p><p>按理说，推翻爱因斯坦相对论是件轰动全球的科学大事，搞不好还能拿个诺贝尔奖。但这事儿没有受到国际权威学术期刊的关注，只在中国舆论炸开锅，难免让人感觉有些不踏实。</p><p>而且，根据学校官网的获奖论文和主持研究的科研项目，以及李子丰教授的简历来看，他研究的方向和天体物理、相对论等基本不沾边。</p><p><strong>“诺贝尔瓷砖奖总能得吧”</strong></p><div id="innity-in-post"></div><div id="dfp-ad-midarticlespecial">        </div><p>在微博上，“燕大教授推翻相对论可信吗？”的话题迅速引起广泛热议，其中以冷嘲热讽居多。</p><p>“建议设诺贝尔笑话奖”“诺贝尔瓷砖奖总能得吧”“不够彻底啊，你咋不把牛顿力学也推翻呢”“牛顿：你不要过来啊！”“很好，下一步就是用老庄思想找外星人”。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/ae.jpg?rwsvOgtHnFlFJO0QPU0CDwSoO1HwwEzJ&itok=4iOji0Nw" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>相对论的创立​者爱因斯坦。（互联网）</figcaption></figure>                  <p>最具争议性的是，李子丰的研究已获推荐入选2021年度河北省科学技术奖，也就是得到了官方的相当认可。网民质疑，“这是道德的沦丧还是学术腐败？” 河北省科技厅则回应称，目前该项目仅处于提名阶段，此后还会有多重评审程序。</p><p>据新京报贝壳财经报道，昨天（22日）登录河北教育厅官网后发现，“河北省教育厅推荐2020年度河北省科学技术奖项目公示”相关内容已经无法找到。李子丰在回应媒体询问时则说，“鉴于我太累，领导让我休息，学术问题也暂不讨论”。</p><p>根据此前媒体披露的官网原文，引发争议的研究项目名称为“坚持唯物主义时空质能观 发展牛顿物理学”，项目简介称，该项目由李子丰一人完成，公示期为2021年6月10日至6月16日。</p><p>研究项目名为“坚持唯物主义时空质能观 发展牛顿物理学”；项目特色是“属于坚持马克思主义哲学的、颠覆性创新的、非共识的理论物理项目”；研究内容则包括“哲学对物理学的指导作用，意识的来源，唯物主义时空质能观，光的本性和传播规律”等。</p><p>面对“哗众取宠”“闹剧一场”等批评声浪，李子丰说，该项目还未立项，报奖的主要目的是“宣传真理”，获奖是小概率事件。</p><p>他信心满满地说：“国家号召颠覆性创新，所谓颠覆性创新就是颠覆台上的理论，才叫颠覆性创新，否则不叫颠覆性创新，我的理论研究认为现在的理论（爱因斯坦相对论）是错误的。”</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/lizifeng1.jpg?LsroUg3mD7OSIKYFwW5PrL.7OQRg894M&itok=e9uIuH_C" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>燕山大学研究员李子丰（互联网）</figcaption></figure>                  <p>至于是否有严谨的科学论证或证据，李子丰称“当然有”，指其现有研究就是以唯物主义作为研究基础，论点就是“实事求是”。</p><p><strong>用哲学裁定科学？</strong></p><p>既是“唯物主义”，又是“马克思主义哲学”、“颠覆性创新”，让人听了很迷糊。是用马克思主义哲学理论去推翻爱因斯坦相对论吗？</p><p>不少评论文章指出，没有数学公式和逻辑推导、没有任何实验的计算，就想要推翻相对论，这不符合自然科学的研究方式，更不符合马克思主义的哲学传统，不仅体现出李子丰物理知识水平的匮乏，还表现出他缺乏实事求是的科学精神。</p><p>北京商报题为“推翻相对论，反智闹剧何时休”的文章直批，经过唯物主义推演的科学论证，却被带有唯心主义的哲学所推翻，“不可谓不荒谬”，并指这样的研究还能登上奖项目录，“其中的草率多少值得深追一番”。</p><p>《新京报》题为“大学教授称‘推翻相对论’，一场哗众取宠的闹剧”的文章则指出，李子丰的项目不是严肃的学术研究项目还未立项，就开始申报评奖，“摆明了就是一场蓄意的炒作”。 </p><p>也有评论指出，从李子丰的表述可看出浓浓的政治味道，但他可能既不懂政治，也不太懂科学，张口就来的样子，动辄要重塑世界的宏愿，倒像是一个典型的“民科”。</p><p><strong>反相对论“民科”先锋</strong></p><p>对于“民科”一词，并没有明确的定义。据《深圳商报》报道，从字面来看，可以理解成民间科学爱好者。相对于传统主流的科研者，他们似乎显得有些“另类”，由此引发的争议和质疑也一直不断。</p><p>对一些动辄喜欢提出“突破性理论”的跨界人士，网友习惯给他们戴上“民科”的帽子。因为这些人的所谓理论，通常经不起相关领域同行的推敲。</p><p>中国网民这几天陆续挖出李子丰身为“民科”先锋的往事。</p><p>他在2017年卷入了“民科集体诉果壳网”一案中，该事件后来被认为是民科诉讼第一案。当时，李子丰和另两人认为“民科”受到果壳网发布文章的侮辱和霸凌，把对方告上法庭，但案件最后以李子丰等人败诉收场。</p><p>在与果壳网的争端中，多名燕山大学学生向校方举报李子丰的行为。同年6月15日，燕山大学发布“关于给予李子丰严重教学事故的通报”，指李子丰曾因在其承担的《钻井工程》课程教学过程中占用大量时间讲授与教学无关的内容，如发表反相对论、反霍金、反阿波罗登月等言论，三年内停止李子丰所有本科教学工作。</p><p>李子丰与相对论的爱恨情仇可追溯至2000年或更早。据称，他在2000年开始研究该项目，并在2008年发表的一篇物理论文中提出了相关论据，也曾多次向有关部门申请立项，但未予立项。</p><p>2009年，中国科协学会服务中心曾回应过“李子丰对狭义相对论的建议信”，委婉表达了“观点尚待验证”。但李子丰显然还未放弃，12年后继续挑战相对论。</p><p><strong>“对伪科学至少还要斗100年”</strong></p><p>李子丰与爱因斯坦杠上的最新事件让人想起了“熟蛋返生孵小鸡”、“新基因编辑技术”和“水变油”等旧闻。</p><p>2016年5月，河北科技大学副教授韩春雨在国际知名科学期刊《自然-生物技术》（Nature Biotechnology）刊登论文，称发明了一种新的基因编辑技术，立即轰动全球。有中国媒体更称韩春雨的研究成果“有望摘取诺贝尔奖”，风头一时无两。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/hanchunyu.jpg?OcGT9Dbrlw7_zpMkoqg2UBklTANSZQ6l&itok=xEEDyLOP" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>2016年5月，河北科技大学副教授韩春雨在国际顶级期刊发文，称发明了一种新的基因编辑技术，但事隔一年多后主动撤回论文。（央视新闻视频截图）</figcaption></figure>                  <p>不过，韩春雨研究成果自公布后，不断被质疑其试验数据无法复制。陷入争议漩涡一年多后，《自然-生物技术》宣布韩春雨研究团队撤回论文。先有个韩春雨，后来个李子丰，因同样是在河北的大学，有网民为此调侃“果然燕赵多人才呀！” ​</p><p>追溯至上世纪八十年代，哈尔滨司机王洪成在1984年3月宣布发明“水变油”，并号称是“中国的第五大发明”。据中国媒体报道，王洪成对外宣称，他研发出一种特殊的母液，只要将母液按照1比100000的比例放入水中混合后，即可制成“水基燃料”。</p><p>这种母液后来也称作“洪成基液”。王洪成自称是一种水基燃料的膨化剂，一点就燃，热值比普通的柴油和汽油高，成本很低，因为由水直接转化为油，燃烧后对环境没有污染，具有一定的环保意义。</p><p>王洪成在各地以表演方式推广“水变油”，日渐成名。据报道，他还以大规模生产为幌子与多家企业合作，诈骗金额不下4亿元人民币，使多家企业损失惨重。</p>      <figure class="image-placeholder "><img alt class src="https://static.zaobao.com/s3fs-public/styles/inline_medium/public/2021-06/chick_and_egg.jpg?Vgc8QARHyzxAQwAlOR3ijg3iBlogljfb&itok=bgOWmUC9" pinger-seen="true" referrerpolicy="no-referrer">        <figcaption>今年4月，一篇题为《熟鸡蛋变成生鸡蛋（鸡蛋返生）-孵化雏鸡的实现报告》的论文在中国网络上炸开了锅。（互联网）</figcaption></figure>                  <p>今年4月，一篇“熟蛋返生孵小鸡”的奇文刊发于《写真地理》杂志引发热议，第一作者为郑州市春霖职业培训学校校长郭萍。</p><p>该文声称，学生利用“超心理意识能量方法”，使煮熟的鸡蛋变成生鸡蛋并孵化小鸡。在引起舆论强烈质疑后，论文作者在压力下辞职，涉事培训学校被停业整改，刊发论文的杂志也停刊整顿处理。</p><p>在“熟蛋返生”冒出后，《科普时报》今年5月发布题为 “对伪科学至少还要斗100年”的文章，讲述中国科学界、新闻界的有识之士多年来，在面对各种伪科学甚嚣尘上时，联手站在维护科学尊严斗争的前列而打出组合拳。文章引述诺贝尔奖获得者李政道的话说：科学不特异，特异非科学。</p><p>就如好些分析文章所说，科学的进步是在证实和证伪的较量之间滚动前进，但如果所谓研究已偏离科学轨道，反而带上“民科”的迷惑色彩，那就不值得一提了。舆论也纷纷指出，“民科”也必须接受严格的学术甄别，不能成为伪科学横行的骗术，更不能催生学术不端与投机取巧，希望李子丰早日通过严谨的科学论证，分享其学术成果。</p><p>李子丰本人曾说过，“反对我的人，会把我的名字倒着念”。</p><p>科学与伪科学的斗争不会停止，相信还会有更多“李子丰们”和“反‘疯子李’们”的较量陆续出现。</p>
