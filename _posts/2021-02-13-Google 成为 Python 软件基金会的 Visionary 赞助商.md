[Google 成为 Python 软件基金会的 Visionary 赞助商](https://www.solidot.org/story?sid=66925)
------

<div>
Google <a href="https://pyfound.blogspot.com/2021/02/welcoming-google-as-visionary-sponsor.html">成为</a> Python 软件基金会最高等级的赞助商：Visionary Sponsor。Python 软件基金会的<a href="https://www.python.org/psf/sponsorship/sponsors/">赞助商等级</a>包括：Associate Sponsor，Participating Sponsor；Partner Sponsor；Contributing Sponsor；Maintaining Sponsor；Sustainability Sponsor 以及 Visionary Sponsor。Visionary Sponsor 意味着 Google 不仅将 Python 用于内部开发，而且将 Python 作为产品的重要组成部分。Google 的赞助费用将被用于资助 CPython 开发，改进供应链安全。Google 从 2010 年起就是 Python 软件基金会的赞助商。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
