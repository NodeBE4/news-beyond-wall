[GitHub CLI 1.0 发布](https://www.solidot.org/story?sid=65576)
------

<div>
GitHub 的命令行工具 CLI <a href="https://github.blog/2020-09-17-github-cli-1-0-is-now-available/">发布了 1.0 正式版本</a>。GitHub CLI 允许开发者在终端内直接操作 GitHub 的各项功能，如 pull requests 和 issues，减少上下文切换，专注于工作。GitHub CLI 1.0 让开发者可以在终端完成整个开发到发布的工作流；允许开发者为任意命令创建别名，能直接访问  GitHub API，等等。<p><img src="https://img.solidot.org//0/446/liiLIZF8Uh6yM.jpg" height="120" style="display:block"/></p>
</div>
